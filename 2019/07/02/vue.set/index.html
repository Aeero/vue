Vue.set

Vue2 如何使用Vue.set?

output : error

Vue 不允許動態添加根級響應式屬性。

不能在直接data上增加屬性，可以在data裡的對象上增加屬性。
例如:Vue.set(vm.someobj, {a,”b”})
